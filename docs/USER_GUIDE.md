## 1. Вход в систему

1. Откройте приложение
2. На экране входа введите **Email** и **Пароль**
3. Нажмите **Войти**
4. При необходимости восстановить пароль нажмите **Забыли пароль?** и введите email

> **Примечание:** При потере соединения приложение показывает ранее загруженное расписание из кэша.

## 2. Роль: Студент

### Главный экран (Дашборд)
- Отображает среднюю посещаемость и средний балл
- Диаграмма посещаемости за период
- Последние объявления
- Последние оценки

### Расписание
- Список занятий сгруппирован по дням недели
- Для каждого занятия показаны: предмет, время, аудитория, преподаватель
- Текущий день выделен автоматически

### Моя посещаемость
- Круговая диаграмма распределения статусов (Присутствовал / Отсутствовал / Опоздал / Уваж. причина)
- Процент посещаемости
- История посещений с датами и статусами

### Мои оценки
- Средний балл по всем предметам
- Разбивка оценок по дисциплинам
- Цветовое кодирование: зелёный (5), синий (4), жёлтый (3), красный (2)
- Тип работы для каждой оценки (текущая, контрольная, экзамен и др.)

### Моя группа
- Информация о группе: название, специальность, год поступления
- Карточка куратора с контактами
- Полный список студентов группы

### Объявления
- Список всех объявлений с закреплёнными вверху
- Нажмите на объявление для просмотра полного текста

## 3. Роль: Преподаватель

### Главный экран
- Сводная статистика: количество занятий, групп, студентов
- Последние объявления

### Расписание
- Персональное расписание преподавателя с группами и аудиториями

### Мои группы
- Список групп и дисциплин преподавателя
- Для каждой дисциплины доступны три действия:
  - **Посещ.** — перейти к отметке посещаемости
  - **Журнал** — открыть журнал оценок
  - **Стат.** — просмотреть статистику группы

#### Отметка посещаемости
1. Выберите занятие из выпадающего списка
2. Установите дату (по умолчанию — сегодня)
3. Для каждого студента выберите статус: ✓ присутствует / ⏰ опоздал / ✗ отсутствует / 📄 уваж. причина
4. Используйте кнопки **Все присутствуют / Все отсутствуют** для быстрой расстановки
5. Нажмите **Сохранить** в правом верхнем углу

#### Журнал оценок
- Список студентов с их оценками по дисциплине
- Средний балл студента выводится справа от имени
- Для добавления оценки нажмите **+** рядом с именем студента:
  1. Выберите значение (2–5)
  2. Выберите тип работы
  3. Нажмите **Сохранить оценку**

#### Статистика группы
- Вкладка **Посещаемость**: круговая диаграмма и процент посещаемости
- Вкладка **Успеваемость**: топ-3 лидера и рейтинг всех студентов с прогресс-баром

## 4. Роль: Куратор

### Дашборд группы
- Карточка группы: название, специальность, количество студентов
- Быстрые кнопки перехода: Студенты / Посещаемость / Успеваемость
- Сводная диаграмма посещаемости

### Посещаемость группы
- Круговая диаграмма со статусами
- Оценка посещаемости: хорошая (≥85%), удовлетворительная (≥70%), низкая (<70%)
- Рекомендации куратору по результатам

### Успеваемость группы
- Средний балл группы
- Топ-3 отличника с медалями
- Полный рейтинг студентов с прогресс-барами

### Список студентов
- Поиск по имени или email
- Нажмите на студента для просмотра детальной информации

## 5. Роль: Администратор

### Пользователи
- Таблица всех пользователей системы
- Поиск по имени/email, фильтрация по роли
- Создание нового пользователя: нажмите **+**
- Редактирование/удаление через меню (⋮) в строке пользователя

### Группы и специальности
- Вкладка **Группы**: управление учебными группами
- Вкладка **Специальности**: управление специальностями
- FAB (+) создаёт новую запись в активной вкладке

### Дисциплины
- Список всех дисциплин с поиском
- Код дисциплины и описание
- Создание/редактирование/удаление

### Расписание
- Полное расписание с фильтром по группе
- Добавление занятия: группа, предмет, преподаватель, день, время, аудитория

### Объявления
- Управление всеми объявлениями
- Выбор аудитории (роли получателей): Студенты, Преподаватели, Кураторы, Администраторы
- Закрепление важных объявлений

### Отчёты
1. Выберите группу
2. Выберите тип: **Посещаемость** или **Успеваемость**
3. Нажмите **Сформировать отчёт**
4. Просмотрите результаты в виде диаграмм и таблиц

### Аналитика
- Сводная статистика по колледжу: количество студентов, преподавателей, групп, дисциплин
- Диаграмма распределения пользователей по ролям
- Общий показатель посещаемости
- Средний балл по колледжу

## 6. Общие функции

### Профиль
- Просмотр личных данных
- Кнопка смены пароля
- Выход из системы

### Настройки
- **Тёмная тема** — переключение светлого/тёмного режима
- **Язык интерфейса** — Русский / Қазақша / English
- **Push-уведомления** — включение/отключение
- **Выйти из системы** — с подтверждением

### Офлайн-режим
При отсутствии интернета приложение отображает ранее загруженное расписание из кэша (актуально 1 час).
